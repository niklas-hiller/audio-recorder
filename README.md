# audio-recorder
Audio Recorder for Machine Learning Project of University

## Tested With
* python 3.10
* Windows 11
* Intel CPU
* Nvidia GPU

## Installation
```powershell
1 | python -m venv venv
2 | venv/Scripts/Activate.ps1
3 | pip install kivy
4 | pip install pipwin
5 | pipwin refresh
6 | pipwin install PyAudio
7 | pip install numpy
8 | python main.py
```
